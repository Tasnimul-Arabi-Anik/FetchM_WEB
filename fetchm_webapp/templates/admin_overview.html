Catalog Policy

Catalog control

Build missing TSVs controls first-time TSV materialization. Refresh existing TSVs controls scheduled rebuilds for taxa that already have a managed TSV. Pausing refresh keeps ready taxa available without requeueing them.

Discovery Policy

Discovery control

Use Paused when you want all worker capacity to stay on TSV generation for release. A single genus can remain in syncing for a long time if its NCBI genome metadata query is especially large.
