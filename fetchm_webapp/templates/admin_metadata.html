Build behavior

Build missing controls first-time metadata generation for taxa that do not have metadata yet. Refresh existing controls scheduled rebuilds for taxa that already have metadata. Genus metadata is built directly from the managed taxon TSV. Species metadata first tries to derive from an already-built genus dataset; if that is unavailable, it falls back to a direct metadata build for that species TSV. Managed builds now write only ncbi_dataset_updated.tsv and a comprehensive ncbi_clean.csv under the selected taxon directory.
